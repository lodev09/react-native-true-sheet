# Usage

1. Import `TrueSheet`.

```tsx
import { TrueSheet } from "@lodev09/react-native-true-sheet"
```

2. Define `TrueSheet` inside any component and attach a `ref` to it.
3. Control the sheet by invoking the available `ref` [methods](/reference/methods).

```tsx {2-2,5-8,11-14,18-25}
export const App = () => {
  const sheet = useRef<TrueSheet>(null)

  // Present the sheet âœ…
  const present = async () => {
    await sheet.current?.present()
    console.log('horray! sheet has been presented ðŸ’©')
  }

  // Dismiss the sheet âœ…
  const dismiss = async () => {
    await sheet.current?.dismiss()
    console.log('Bye bye ðŸ‘‹')
  }

  return (
    <View>
      <Button onPress={present} title="Present" />
      <TrueSheet
        ref={sheet}
        sizes={['auto', 'large']}
        cornerRadius={24}
      >
        <Button onPress={dismiss} title="Dismiss" />
      </TrueSheet>
    </View>
  )
}
```

Simple, right? Head over to [`props`](/reference/props) to learn more about configuring your sheet.
