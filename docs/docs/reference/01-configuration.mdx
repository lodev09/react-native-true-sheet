Configuration props available for `TrueSheet`. Extends `ViewProps`.

```tsx {2-5}
<TrueSheet
  ref={sheet}
  detents={['auto', 0.8, 1]}
  backgroundColor="#696969"
  // ...
>
  <View />
</TrueSheet>
```

## `ref`

We use `ref` to reference our sheet and call the [imperative methods](/reference/methods). Learn more about refs [here](https://react.dev/learn/referencing-values-with-refs).
