`TrueSheet` has a main prop called [`detents`](/reference/props#detents) which allows you to define the detents that the sheet can support. This is an array of [`SheetDetent`](/reference/types#sheetdetent) that supports values like `"auto"` or fractional numbers (0-1).

In some cases, you may want to resize the sheet programmatically for a better experience.

<img alt="resizing" src={resizing} width="300"/>

## How?

### Resize Programmatically

Define the sheet and use the [`resize`](/reference/methods#resize) method.

```tsx {2-2,6-6,11-13}
const App = () => {
  const sheet = useRef<TrueSheet>(null)

  const resize = async () => {
    // Resize to 69%
    await sheet.current?.resize(1)
    console.log('Yay, we are now at 69% ðŸ’¦')
  }

  return (
    <TrueSheet name="resizing-sheet" ref={sheet} detents={['auto', 0.69, 1]}>
      <Button onPress={resize} title="Resize" />
    </TrueSheet>
  )
}
```

:::tip
You can also do it globally using the related [global method](/reference/methods#global-methods).

```tsx
TrueSheet.resize('resizing-sheet', 1)
```
:::

:::info
`detents` can only support up to 3 detents. **_collapsed_**, **_half-expanded_**, and **_expanded_**.
:::


### Listening to Detent Change

For some reason, if you want to get the resolved detent, you can listen to detent changes by providing the [`onDetentChange`](/reference/props#ondetentchange) event.

The event comes with the [`DetentInfo`](/reference/types#detentinfo) that you can use to get the detent `value`.

```tsx {9-11,17-17}
const App = () => {
  const sheet = useRef<TrueSheet>(null)

  const resize = async () => {
    // Resize to 69%
    await sheet.current?.resize(1)
  }

  const handleDetentChange = (e: DetentChangeEvent) => {
    console.log(e.nativeEvent.index, e.nativeEvent.value) // Do whatever you need from this detent âœ…
  }

  return (
    <TrueSheet
      ref={sheet}
      detents={['auto', 0.69, 1]}
      onDetentChange={handleDetentChange}
    >
      <Button onPress={resize} title="Resize" />
    </TrueSheet>
  )
}
```

:::info
The event will also trigger when the user drags the sheet into a detent.
:::
